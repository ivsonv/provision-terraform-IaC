### Setup
* terraform version = `v1.8.5`

```sh
terraform init
```

set credentials

```sh
export AWS_ACCESS_KEY_ID=YOUR_ACCESS_KEY_ID
export AWS_SECRET_ACCESS_KEY=YOUR_ACCESS_KEY
```

configurations in terminal
* aws configure confirm `access_key`, `secret_key` and `region`
